# Machine Learning

Computer vision and NLP projects covering image classification, object detection, and text classification.

| Notebook | Description |
|---|---|
| `Bone_Fracture_Detection.ipynb` | Transfer learning with **EfficientNetB0** to classify X-ray images as fractured/not fractured. Dataset sourced via `kagglehub`. |
| `Brain_Tumor_Detection.ipynb` | **YOLO** (Ultralytics) object detection model trained on axial MRI scans to localize and classify brain tumors. |
| `Mental_Health_Text_Classification__NLP_.ipynb` | NLP pipeline using **NLTK** — tokenization, stopword removal, lemmatization, POS tagging, and word clouds — to classify mental health-related text statements. |

**Tools:** `TensorFlow/Keras`, `EfficientNetB0`, `Ultralytics YOLO`, `OpenCV`, `scikit-learn`, `NLTK`, `WordCloud`

> Note: Some notebooks reference local file paths (e.g., `C:\Users\...`) or Kaggle dataset downloads — update these paths/download the datasets before re-running.
