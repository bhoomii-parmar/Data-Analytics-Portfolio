# Excel & Spreadsheets — Retail Sales & Inventory Dashboard

An end-to-end Excel analytics project for a fictional retail brand ("The Souled Store" practice dataset), covering the full retail data lifecycle — raw sales data, data cleaning, inventory tracking, and an interactive dashboard.

### File
- `Retail_Sales_and_Inventory_Dashboard.xlsx`

### What's inside
| Sheet | Purpose |
|---|---|
| `Orders` / `Orders_Raw` / `Orders_cleaned` | Raw and cleaned order-level sales data (order ID, product, category, revenue, channel, payment mode, order status). |
| `Product_Master` | Product catalog with cost price, MRP, launch date, and supplier. |
| `Inventory` | Stock levels by SKU and warehouse, including reorder levels and damaged stock. |
| `Purchase_Orders` / `Suppliers` / `Warehouses` / `Stores` | Supply-chain reference tables. |
| `Monthly_Targets` | Sales targets for performance tracking. |
| `MIS_REPORTS` | Management Information System summary tables built with `SUMIFS`/formula-based reporting (revenue by category, data quality checks, etc.). |
| `Pivot` | PivotTable-based analysis. |
| `Dashboard` | Interactive dashboard with KPIs — Total Revenue, Total Profit, Total Orders, Average Order Value, Profit Margin %, and Return Rate %. |
| `Tasks` / `merge` | Supporting/working sheets used during analysis. |

### Skills demonstrated
Data cleaning, formula-based reporting (`SUMIFS`, lookups), PivotTables, KPI dashboard design, inventory & supply chain analysis, and MIS reporting in Excel.
