## 📊 Power BI
| Project | Description |
|---|---|
| **Bank Customer Analytics 2025** | Churn analysis dashboard exploring churn rate by geography, age, gender, credit score, and number of products for a bank's customer base. |
| **Chronic Kidney Disease Analysis Dashboard** | Clinical dashboard tracking CKD event rates against risk factors like hypertension, diabetes, smoking, eGFR, and creatinine levels. |
| **Country-Wise Sales Performance** | Multi-page retail sales report covering top/bottom countries and products, Pareto analysis, RFM customer segmentation, and monthly sales trends. |
