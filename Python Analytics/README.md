# Python Analytics

Exploratory data analysis, statistics, and data cleaning notebooks.

| Notebook | Description |
|---|---|
| `01_data_cleaning.ipynb` | Loads and cleans a Mutual Fund dataset — standardizes column names and prepares data for analysis using pandas. |
| `mean__reg_and_corr.ipynb` | Statistical analysis of survey data: descriptive statistics, correlation matrix (heatmap), and ANOVA (F-test) to compare group means, using `statsmodels`, `scipy`, and `seaborn`. |
| `Mutual_fund_investment_plan.ipynb` | Downloads and analyzes fund/stock performance data via `yfinance`, with visualizations using matplotlib/seaborn. |
| `COVID_Tests_Data_Analysis.ipynb` | Exploratory analysis of Indian state-wise COVID testing data — sorting, filtering, and conditional analysis with pandas. |

**Tools:** `pandas`, `numpy`, `matplotlib`, `seaborn`, `statsmodels`, `scipy`, `yfinance`
